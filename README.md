# Manga-to-Kindle-Converter
cbz conversion tool that optimizes specifically manga for reading on popular kindles and other e-ink readers.


this is a tool to convert manga/comics for the best reading experience on an e-reader device.


there are several presets built in, and the option to input your own custom resolution

***more presets coming soon***


there are a few basic color options so that the tool is as simple to use as possible.


lets you batch import several cbz files and select an output folder.


this tool also lets you just put in a folder full of images and itll make a cbz out of it
